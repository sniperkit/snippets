# Memory model

```
            Header

            +-----------------------+
            |  0x5f 0x4e 0x41 0x4e  |
  *Magic*   +-----------------------+
"_NANOKV_"  +-----------------------+
            |  0x4f 0x4b 0x56 0x5f  |
            +-----------------------+

            Allocator

            +-----------------------+
            |  Used bytes           |
            +-----------------------+
            +-----------------------+
            |  Free bytes           |
            +-----------------------+

            Key-Value
            +-----------------------+
            | Key hash (MurmurHash2)|
            +-----------------------+
            +-----------------------+
            | Key size  || Val size |
            +-----------------------+
            +-----------------------+
            | Next kv offset (bytes)|
            +-----------------------+
            +-----------------------+
            | Key data (max 255B)   |
            +-----------------------+
            +-----------------------+
            | Val data (max 255B)   |
            +-----------------------+
```
